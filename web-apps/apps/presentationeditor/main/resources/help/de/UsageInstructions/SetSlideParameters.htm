Bestimmen Sie die Folienparameter

Um Ihre Präsentation zu personalisieren, können Sie ein Thema und Farbschema für die ganze Präsentation wählen sowie die Füllung des Hintergrunds oder das Layout für jede einzelne Folie ändern. Sie können auch Übergänge zwischen den Folien anwenden.

  • Die Themen erlauben Ihnen, das Design der Präsentation schnell zu ändern, besonders die Hintergrundansicht der Folie, vorgegebene Schriftarten für den Titel und Text und das Farbschema, das für die Elemente der Präsentation genutzt wird. Um ein Thema für die Präsentation zu wählen, klicken Sie auf das gewünschte vorgegebene Thema auf der rechten Seite der oberen Symbolleiste. Das gewählte Thema wird auf alle Folien angewandt, wenn Sie vorher keine bestimmten Folien gewählt haben.

  • Die Farbschemata beeinflußt vorgegebene Farben, die für die Elemente der Präsentation genutzt werden (Schriftarten, Linien, Füllung usw.), und erlauben Ihnen, die Farbenkonsistenz in der ganzen Präsentation zu behalten. Um ein Farbschema für die Präsentation zu ändern, klicken Sie aufs Symbol Farbschema ändern auf der oberen Symbolleiste und wählen Sie das gewünschte Schema aus dem Listenmenü. Das gewählte Schema wird auf alle Folien angewandt.

  • Um die Foliengröße für die ganze Präsentation zu ändern, klicken Sie auf das Symbol Foliengröße wählen auf der oberen Symbolleiste und wählen Sie die gewünschte Option im Listenmenü. Sie können wählen:
    • eine der Voreinstellungen: Standard (4:3) oder Breitbildschirm (16:9),
    • die Option Erweiterte Einstellungen, die das Fenster Einstellungen der Foliengröße öffnet, wo Sie eine der gewählten Voreinstellungen wählen oder eine benutzerdefinierte Größe durch die Angabe der Breite und Höhe festlegen können.

      Die verfügbaren Voreinstellungen sind Standard (4:3), Breitbildschirm (16:9), Breitbildschirm (16:10), Letter Paper (8.5x11 in), Ledger Paper (11x17 in), A3 Paper (297x420 mm), A4 Paper (210x297 mm), B4 (ICO) Paper (250x353 mm), B5 (ICO) Paper (176x250 mm), 35 mm Slides, Overhead, Banner.

  • Um die Füllung des Hintergrunds zu ändern:
    1. In der Folienliste links wählen Sie die Folien, auf die Sie die Füllung anwenden möchten. Oder klicken Sie an jeder leeren Stelle im Bearbeitungsbereich der aktuellen Folie, um den Füllungstyp für diese einzelne Folie zu ändern.
    2. Auf der Registerkarte Folieneinstellungen im rechten Seitenbereich wählen Sie die notwendige Option:
      • Farbfüllung - wählen Sie diese Option, um die gewünschte solide Farbe zu bestimmen, die auf die gewählten Folien angewandt wird.
      • Füllung mit Farbverlauf - wählen Sie diese Option, um die Folie mit einem sanften Übergang von einer Farbe zu einer anderen zu füllen.
      • Bild oder Textur - wählen Sie diese Option, um ein Bild oder eine vorgegebene Textur als Folienhintergrund festzulegen.
      • Muster - wählen Sie diese Option, um die Folie mit dem zweifarbigen Design, das aus regelmässig wiederholten Elementen besteht, zu füllen.
      • Keine Füllung - wählen Sie eine Option, wenn Sie keine Füllung verwenden möchten.

      Weitere Informationen über diese Optionen finden Sie in der Sektion Füllen Sie die Objekte und wählen Sie die Farben.

  • Übergänge helfen Ihre Präsentation dynamischer zu machen und Aufmerksamkeit Ihres Publikums zu halten. Um einen Übergang anzuwenden:
    1. wählen Sie die Folien, auf die Sie einen Übergang anwenden möchten,
    2. wählen Sie einen Übergang im Listenmenü Effekt in der Registerkarte Folieneinstellungen,
    3. stellen Sie die Eigenschaften des Übergangs ein: wählen Sie die Variante, die Dauer des Übergangs und die Art des Folienwechsels,
    4. klicken Sie auf den Button Auf alle Folien anwenden, falls Sie denselben Übergang auf alle Folien in der Präsentation anwenden möchten.

      Weitere Informationen über diese Optionen finden Sie in der Sektion Übergänge anwenden.

  • Um das Folienlayout zu ändern:
    1. In der Folienliste links wählen Sie die Folien, auf die Sie das neue Layout anwenden möchten.
    2. Klicken Sie aufs Symbol Folienlayout ändern auf der oberen Symbolleiste.
    3. Wählen Sie das gewünschte Layout im Menü.

      Alternativ können Sie mit der rechten Maustaste auf die gewünschte Folie in der Liste links klicken, die Option Layout ändern im Kontextmenü wählen und das gewünschte Layout bestimmen.

      Hinweis: Momentan sind die folgenden Layouts verfügbar: Title, Title and Object, Section Header, Two Objects, Two Texts and Two Objects, Title Only, Blank, Title, Object and Caption, Picture and Caption, Vertical Text, Vertical Title and Text.
