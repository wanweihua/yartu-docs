Insérer un graphique

Pour insérer un graphique dans une feuille de calcul,

  1. sélectionnez la plage de cellules qui contiennent les données que vous souhaitez utiliser pour le graphique,
  2. cliquez sur l'icône Insérer un graphique sur la barre d'outils supérieure,
  3. dans la fenêtre ouverte spécifiez les paramètres du graphique :
    • Type de graphique, style, plage de données
      Sélectionnez un type de graphique que vous souhaitez insérer : Ligne, Colonne, Barre, Zone, Secteur, Point, Stock et son style. Vérifiez la plage de cellules, modifiez-la, si nécessaire, et choisissez la façon d'organisation des données. Vous pouvez choisir soit Série en ligne soit Série en colonne à utiliser sur l'axe X.

    • Éléments de graphique, légende de graphique
      Spécifiez le Titre du graphique et les éléments à afficher : Valeurs du graphique, Bordures du graphique, Axes et Quadrillage ainsi que des titres des axes dans les champs correspondants. Si vous souhaitez qu'une légende s'affiche, laissez la case correspondante cochée et indiquez la position de votre légende par rapport à votre graphique en sélectionnant l'option nécessaire à partir de la liste déroulante ci-dessous : En bas, En haut, A droite, A gauche.

  4. une fois que le graphique est ajouté, vous pouvez modifier sa taille et sa position.

Pour changer le type du graphique inséré,

  1. sélectionnez le graphique avec la souris,
  2. cliquez sur l'icône Paramètres du graphique sur la barre latérale droite,

  3. dans la section Modifier le type de graphique ouvrez la liste Type de graphique et sélectionnez le type nécessaire,
  4. ouvrez la liste déroulante Style de graphique à côté du Type de graphique et sélectionnez le style qui vous convient.

Le type et le style du graphique sélectionné seront changés. Si vous avez besoin de modifier les données utilisées pour créer le graphique,

  1. cliquez sur le bouton Modifier les données sur la barre latérale droite,
  2. dans la fenêtre Paramètres du graphique faites toutes les modifications nécessaires,
  3. cliquez sur OK pour appliquer les modifications et fermer la fenêtre.

Pour supprimer le graphique inséré, sélectionnez-le avec la souris et appuyez sur la touche Supprimer.
